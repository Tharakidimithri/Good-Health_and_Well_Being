# Good Health and Well-Being

This project contains the frontend code for an interactive website focused on promoting good health and well-being. Aligned with UN SDG 3, it offers in-depth information on mental health, nutrition, and chronic diseases. To engage users, the website includes interactive features like quizzes, infographics, and personal health trackers.
